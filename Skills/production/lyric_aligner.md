# Lyric Aligner — AI Skill v1.0

**Role:** Runs Whisper speech recognition on the master audio to extract word-level timestamps, maps every lyric section to an exact second range, and corrects any estimated timestamps in the cut list.
**Phase:** Production
**Inputs:** `Sound/master.mp3`, `Lyrics/LYRICS_[project_id].md` (Section 4: SUNO Lyrics), `Assembly/cut_list.json` (for timestamp correction)
**Outputs:** `Lyrics/timestamps_aligned.json`, `Lyrics/timestamps_map.md`
**Tools:** `whisper` (OpenAI), `ffprobe`
**Human Gate:** NO — automated
**Reads from PROJECT_STATE:** `asset_registry.audio.master`, `language`
**Writes to PROJECT_STATE:** `asset_registry.audio.timestamps_json`, `current_stage → creative_design`
**Version History:** v1.0 — Initial release (2026-05-17)

---

## Purpose

Precise lyric timing transforms a generic slideshow into a video where images actually react to the song. This skill locks every section to an exact second range so the creative_designer can build a beat-accurate storyboard, and so the video_editor knows exactly when to cut.

Cost: ~$0.02 per song (OpenAI Whisper API). Negligible — always run this.

---

## Step 1 — Pre-flight Check

```bash
ffprobe -v quiet -show_entries format=duration -of csv="p=0" "Sound/master.mp3"
```

Confirm audio is accessible. Report duration.

If file is missing: `STOP — Sound/master.mp3 not found. Run audio_engineer skill first.`

---

## Step 2 — Run Whisper

```bash
# Recommended: OpenAI API (no local GPU required)
whisper "Sound/master.mp3" \
  --language [language from PROJECT_STATE] \
  --output_format json \
  --output_dir Lyrics/ \
  --word_timestamps true
```

**Language codes:** Russian = `ru`, English = `en`, Spanish = `es`, French = `fr`, German = `de`

If Whisper CLI is not installed:
```bash
pip install openai-whisper
```

Output: `Lyrics/master.json` (word-level timestamps from OpenAI)

---

## Step 3 — Map Lyrics to Timestamp Sections

Parse `Lyrics/master.json`. For each section in `LYRICS_[project_id].md`:
1. Find the first word of the section in Whisper output
2. Find the last word of the section
3. Record `start_time` and `end_time` in seconds

**Handle instrumental sections** (intro, interlude, outro) by measuring the gap between the last word of the previous section and the first word of the next section.

Output `Lyrics/timestamps_map.md`:

```markdown
# Lyric Timestamp Map — [Song Title]
Generated by Whisper | Date: [today]

| Section | Lyrics preview (first 5 words) | Start (s) | End (s) | Duration (s) |
|---|---|---|---|---|
| Intro | *(instrumental)* | 0.0 | 12.4 | 12.4 |
| Verse 1 | "[first 4–5 words of verse 1]..." | 12.4 | 45.2 | 32.8 |
| Chorus 1 | "[first 4–5 words of chorus]..." | 45.2 | 72.0 | 26.8 |
| Verse 2 | "[first 4–5 words of verse 2]..." | 72.0 | 104.5 | 32.5 |
| Bridge | "[first 4–5 words of bridge]..." | 104.5 | 122.3 | 17.8 |
| Chorus 2 | "[first 4–5 words of chorus]..." | 122.3 | 149.1 | 26.8 |
| Outro | *(instrumental)* | 149.1 | 163.0 | 13.9 |

**Total duration:** 163.0s  
**Source:** Whisper word-level alignment (high confidence)
```

Write machine-readable `Lyrics/timestamps_aligned.json`:
```json
{
  "song_title": "[title]",
  "language": "[language]",
  "total_duration_s": 163.0,
  "whisper_confidence": "high",
  "sections": [
    {"id": "intro",    "label": "Intro",    "start": 0.0,   "end": 12.4},
    {"id": "verse_1",  "label": "Verse 1",  "start": 12.4,  "end": 45.2},
    {"id": "chorus_1", "label": "Chorus 1", "start": 45.2,  "end": 72.0},
    {"id": "verse_2",  "label": "Verse 2",  "start": 72.0,  "end": 104.5},
    {"id": "bridge",   "label": "Bridge",   "start": 104.5, "end": 122.3},
    {"id": "chorus_2", "label": "Chorus 2", "start": 122.3, "end": 149.1},
    {"id": "outro",    "label": "Outro",    "start": 149.1, "end": 163.0}
  ]
}
```

## Step 4 — Validate and Correct cut_list.json

If `Assembly/cut_list.json` already exists (written by scriptwriter):

For each scene in the cut list, compare its `start_s` and `end_s` against the Whisper timestamps for the same section.

Flag discrepancies > 3 seconds:
```
⚠️ TIMESTAMP MISMATCH — Verse 2
   Script estimated:  80.0s – 112.0s
   Whisper measured:  72.0s – 104.5s
   Difference:        8.0s
   Action: Updating cut_list.json to Whisper values.
```

Update all scene timestamps in `cut_list.json` to match Whisper output. Report total number of scenes corrected.

---

## Failure Recovery

**Whisper produces poor transcript (non-English / unusual accent):**
- Try `--model large` (more accurate but slower):
  ```bash
  whisper "Sound/master.mp3" --language ru --model large --word_timestamps true --output_format json --output_dir Lyrics/
  ```
- If transcript is still garbled, fall back to manual section timing: ask the human to listen and provide the approximate start/end of each lyric section.

**Audio file is too short or corrupted:**
```bash
ffprobe -v error -show_entries format=duration,size -of json "Sound/master.mp3"
```
Report findings and stop.
