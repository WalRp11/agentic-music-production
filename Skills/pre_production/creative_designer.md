# Creative Designer — AI Skill v1.0

**Role:** Transforms the approved scene list into a final storyboard arrangement — defining visual rhythm, shot sequencing logic, transition types, and intensity mapping — producing the blueprint the video_editor uses to build the timeline.
**Phase:** Pre-Production
**Inputs:** `Lyrics/music-video-script.md`, `Assembly/cut_list.json`, `Lyrics/timestamps_aligned.json`, `Lyrics/LOCATIONS_[project_id].md`, `Lyrics/CASTING_[project_id].md`
**Outputs:** `Assembly/storyboard_arrangement.md`
**Tools:** None (creative reasoning)
**Human Gate:** NO — automated; human already approved the script. This is execution design, not creative direction.
**Reads from PROJECT_STATE:** `decisions.video_aspect_ratio`, `decisions.color_grade_style`
**Writes to PROJECT_STATE:** `asset_registry.assembly.storyboard_arrangement`, `current_stage → image_generation`
**Version History:** v1.0 — Initial release (2026-05-17)

---

## Purpose

The scriptwriter writes what each scene *is*. The creative designer decides how those scenes *flow together* — rhythm, pacing, transitions, emotional arc. This document is the video_editor's blueprint and removes all sequencing ambiguity from the assembly step.

Without this skill, the video_editor makes arbitrary sequencing and transition decisions that rarely match the emotional intent of the music.

---

## Step 1 — Load All Source Documents

Read in this order (each informs the next):
1. `Lyrics/timestamps_aligned.json` — exact section timing (from Whisper)
2. `Lyrics/music-video-script.md` — scene descriptions and intended emotion
3. `Assembly/cut_list.json` — shot order, durations, scene IDs
4. `Lyrics/LOCATIONS_[project_id].md` — location visual identities and their emotional weight
5. `Lyrics/CASTING_[project_id].md` — character arcs (beginning / middle / end emotion)

---

## Step 2 — Assign Intensity to Each Section

For each lyric section, assign a visual intensity level. Use the emotional arc from the concept brief as your guide.

| Intensity | Description | Shot implications |
|---|---|---|
| 1 — Minimal | Instrumental intro/outro | Wide establishing, very slow pan or static |
| 2 — Quiet | Early verse, reflective | Medium shots, near-static, character observing |
| 3 — Building | Mid-verse, pre-chorus | Mix of wide and medium, subtle push-in |
| 4 — Peak | Chorus, bridge climax | Close-ups, more frequent cuts (8–6s per cut) |
| 5 — Release | Post-chorus, outro resolve | Pull-back, wide, return to stillness |

**Rule:** Intensity must follow a curve — don't spike to 4 before building through 2 and 3.

---

## Step 3 — Design Transition Logic

Assign a transition type to each scene boundary:

| Transition | When to use |
|---|---|
| Hard cut | Default — most scene changes; keeps energy |
| Dissolve (0.8s) | Same location, different moment; emotional bridge |
| Fade to black (1.2s) | Between acts (verse → chorus first time); time passing |
| Match cut | Same shape or subject across adjacent scenes — powerful when used once |
| No transition | Never leave a scene without specifying one |

**Rules:**
- Maximum 2 dissolves per video — overuse destroys impact
- Match cuts should be planned intentionally, not discovered in the edit
- The final scene before the first chorus always gets a hard cut (creates impact)

## Step 4 — Write Storyboard Arrangement

Output `Assembly/storyboard_arrangement.md`:

```markdown
# Storyboard Arrangement — [Song Title]
> Generated by Creative Designer v1.0 | [Date]
> This document drives the video_editor. Do not modify manually — re-run creative_designer if changes are needed.

---

## Overview
Total scenes: [N]
Total duration: [X]s
Average scene length: [X]s
Emotional arc: [Beginning mood] → [Middle mood] → [End mood]

---

## Scene Sequence

| # | Scene ID | Location | Intensity | Duration | Transition OUT | Emotional note |
|---|---|---|---|---|---|---|
| 1 | c01 | [Location A name] | 1 | 10s | Hard cut | Sets world — [opening mood] |
| 2 | c02 | [Location B name] | 2 | 10s | Dissolve | [Character / subject] in [state] |
| 3 | c03 | [Location A name] | 2 | 10s | Hard cut | [First emotional beat] |
| ... | | | | | | |
| [N] | c[N] | [location] | 5 | 10s | Fade to black | Resolution |

---

## Emotional Arc Map

| Time range | Section | Intensity | Visual focus |
|---|---|---|---|
| 0:00–0:12 | Intro | 1 | Environment only — no character yet |
| 0:12–0:45 | Verse 1 | 2 | [Protagonist], [opening emotional state] |
| 0:45–1:12 | Chorus 1 | 4 | Emotional peak — close-ups, faster cuts |
| ... | | | |

---

## Transition Schedule

| After scene | Into scene | Transition type | Duration |
|---|---|---|---|
| c01 | c02 | Hard cut | — |
| c08 | c09 | Dissolve | 0.8s |
| ... | | | |

---

## Notes for Video Editor
- [Any special instructions — e.g. "c14 should land exactly on the word 'пути' — see timestamps_aligned.json"]
- [Any beat-sync requests — e.g. "The piano hit at 72.0s should coincide with the cut to c16"]
```
